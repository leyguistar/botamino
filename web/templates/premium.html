La version lite del bot es una version que corre en un servidor ligero, revisando todos los chats y respondiendo a los comandos.

Como el servidor es mas ligero no tiene todos los comandos, no tiene un registro de los mensajes, no funciona todo el tiempo.

La version pro del bot es una version dedicada exclusiva para uno o varios chats.

  • Es mas rapida que la version lite.
  • Tiene mas comandos.
  • Agradece las donaciones.
  • Tiene un registro de todos los mensajes.
  • Se pueden configurar los permisos de cada usuario.
  • Siempre esta funcionando
